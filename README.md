# Fashion Trend Prediction & Analytics System

## Description
This project uses MongoDB to store and analyze fashion trend data.

## Features
- 15 collections database
- 45 MongoDB queries
- Trend analysis using aggregation
- Popularity score calculation

## Technologies Used
- MongoDB
- MongoDB Query Language (MQL)

## Files Included
- Dataset (JSON)
- Queries file
- Project report

## How to Run
1. Import JSON file into MongoDB
2. Run queries in MongoDB Compass or shell
