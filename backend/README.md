# Backend (Express + MongoDB)

## Setup

1. Copy `.env.example` to `.env`
2. Update `MONGODB_URI` if needed.

## Run

```powershell
npm install
npm run dev
```

API runs on `http://localhost:5000` (default).

## Endpoints

- `GET /users`
- `GET /posts`
- `GET /trends`
- `GET /keywords`
- `GET /regions`
- `GET /trendAnalytics`
- `POST /runQuery`
